Iframe API


How to use?

Each GET method returns the code for the graph. This means you can put it inside the src of a <iframe>.

For example: <iframe src="{YOUR_REQ_HERE}"></iframe>.

The base url for all requisitions is https://exileinfo.xyz/api/.

As a recommendation, You should stick with at least 300px of height, so that things don't get too cramped.


Leagues available:


legacy
Legacy League.

hclegacy
Hardcore Legacy League.

ssflegacy
Solo self found Legacy League.

ssfhclegacy
Solo self found Hardcore Legacy League.
